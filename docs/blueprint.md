# **App Name**: RagInfo

## Core Features:

- PDF Upload and Extraction: Allow users to upload PDF files (max 10MB), extract text content, and store the filename.
- Vector Database Storage: Store extracted text in a vector database for efficient semantic search and retrieval.
- File Management: Enable users to delete uploaded files and associated data from the vector database.
- Chat Interface: Implement a chat interface for users to interact with the AI tool.
- AI Query and Response: Process user chat input, query the vector database, and display responses from the AI tool.
- Chat Entry Management: Allow users to copy AI responses and delete individual chat entries along with the associated AI answers.
- API Key and Model Configuration: Allow users to input their Google AI API key and model (e.g., Google 2.5) and save it for persistent use as a tool for chat operations.

## Style Guidelines:

- Primary color: Dark blue (#224292) for a professional and sophisticated feel.
- Background color: Light blue (#E0F7FA) for a clean and calming interface.
- Accent color: Medium blue (#4FC3F7) to highlight interactive elements and call-to-actions.
- Body and headline font: 'Inter' sans-serif font, for a modern and objective look.
- Use minimalistic line icons in shades of blue to represent different actions and file types.
- Maintain a clean and intuitive layout with a focus on usability, using a grid system to align elements.
- Incorporate subtle animations for user interactions, such as file uploads and chat responses, to enhance user experience.
